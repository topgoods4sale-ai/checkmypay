# CheckMyPay - Virginia Paycheck Calculator

A fast, mobile-friendly Next.js utility site for the keyword: Virginia paycheck calculator.

## Launch free on Vercel
1. Create a free GitHub account.
2. Create a free Vercel account.
3. Create a new GitHub repo named `checkmypay`.
4. Upload these files.
5. In Vercel, click Add New Project > import the repo > Deploy.
6. Use the free `your-project.vercel.app` domain first.

## AdSense
Replace the three placeholder ad boxes in `app/page.tsx` after Google AdSense approves the site.

## Important
This is an estimate calculator. Tax rules and payroll withholding can change.
